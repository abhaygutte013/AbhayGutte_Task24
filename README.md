# Animations & UI Feedback using Tailwind CSS

## Objective

The objective of this project is to demonstrate different animation techniques and interactive UI feedback using Tailwind CSS. The webpage showcases built-in animations, hover effects, loading states, notification badges, transitions, and custom staggered animations to improve user experience.

## Technologies Used

- HTML5
- Tailwind CSS (CDN)
- Font Awesome Icons

## Features

### Header Section

- Responsive page heading
- Subtitle describing the purpose of the webpage

### Built-in Animations

This section demonstrates Tailwind's built-in animation utilities:

- animate-spin
- animate-ping
- animate-pulse
- animate-bounce

Each animation is displayed inside a separate card with its name and description.

### Button Animations

## How to Run

1. Download the project files.
2. Open the folder in Visual Studio Code.
3. Make sure you have an internet connection because Tailwind CSS and Font Awesome are loaded using CDN.
4. Open `index.html` in your browser.
